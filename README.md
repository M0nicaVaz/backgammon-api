## Setup

**Clone the project and access the folder**

```bash
git clone https://github.com/M0nicaVaz/backgammon-api.git && cd backgammon-api
```

**Follow the steps below**

```bash
# Install the dependencies
$ npm or yarn install

# Run the web server
$ npm run dev
```
